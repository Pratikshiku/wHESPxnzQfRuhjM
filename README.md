## Fire Management System Enhanced Version with Message Module Online Learning

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/210caac5f7f74c608e776fa3ef22ff5c/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `layui`, `HTML` / `CSS` / `Javascript` / `bootstrap` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): System Administrator, Recruiter, Student.

**Student Functions:**
1. Login
2. Register
3. My Messages
4. Online Learning
5. My Registration
6. My Admission Ticket
7. Personal Center
8. Secure Logout


**Recruiter Functions:**
1. Login
3. Personal Information Management
4. Registration Management
5. Score Management
6. Admission Ticket Management
7. Message Management
8. Online Learning Management



**Administrator Functions:**
1. Login
2. Personal Information Management and Viewing
3. Registration Management
4. Score Management
5. Candidate Management
6. Admission Ticket Management
7. Administrator Management
